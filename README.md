# GrowthTracker — 儿童成长跟踪

**发育行为评估 + 生长曲线 | Child Growth Tracker**

基于中华人民共和国卫生行业标准开发的儿童成长跟踪工具，供家长记录和追踪孩子的发育与生长状况。

---

## 功能模块

### 发育行为评估（WS/T 580—2017）

- 录入孩子基本信息，按月龄自动匹配评估组
- 支持回溯录入历史发育记录
- 261 个评估项目，覆盖 28 个月龄组 × 5 个能区（大运动、精细运动、语言、适应性、个人-社会）
- 每项附带操作方法和通过要求说明（附录 B）
- 自动评分，生成发育商（DQ）报告

### 生长曲线（WS/T 423—2022）

- 记录身高、体重、头围测量数据，每项可灵活填写
- BMI 自动计算
- SVG 绘制百分位参考曲线（P3 / P25 / P50 / P75 / P97）
- 智能视窗，以当前月龄为中心，横向可滚动
- 支持身高、体重、头围、BMI 四类生长曲线

## 技术栈

- **后端**：Node.js v20 + Express 4
- **前端**：原生 HTML / CSS / JavaScript（无框架依赖）
- **数据存储**：`data/children/{uuid}.json`（原子写入）

---

## 本地部署

### 前置条件

- Node.js v18+

### 步骤

```bash
# 克隆仓库
git clone https://github.com/rangerqu/growthtracker.git
cd growthtracker

# 安装依赖
npm install

# 启动服务
npm start
```

服务默认运行于 `http://localhost:3000`。

### 自定义端口

```bash
PORT=8080 npm start
```

## 标准来源

本项目基于以下卫生行业标准实现：

> 中华人民共和国国家卫生和计划生育委员会. *WS/T 580—2017 0～6岁儿童发育行为评估量表*. 2017.
>
> 中华人民共和国国家卫生健康委员会. *WS/T 423—2022 7岁以下儿童生长标准*. 2022.

标准内容版权归原标准制定机构所有，本项目仅为个人及非商业用途的技术实现。
